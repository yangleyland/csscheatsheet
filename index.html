Display and Positioning Reference

Position Elements

Name Description Image
Static default position
Relative element positioned relative to original position
Fixed an element that stays the same regardless of scrolling (ex: navigation bar)
Absolute removes an element from the document flow

Display Elements

Name Description Image
Block elements take up the full width of the container, height and width can be adjusted
Inline elements take up as little space as possible, take up the line, and height and width cannot be adjusted
Inline-block elements appear next to each other, can have width and height adjusted
